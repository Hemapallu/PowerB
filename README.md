## Project Overview
This project focuses on analyzing customer churn behavior using Power BI.  
The dashboard helps identify key factors influencing customer churn and provides insights that support better customer retention strategies.

The analysis is based on customer demographics, tenure, services, contract types, billing methods, and subscription behavior.

---

## Objectives
- Understand customer churn patterns
- Identify high-risk customer segments
- Analyze the impact of services, contracts, and billing methods on churn
- Provide data-driven insights for business decision-making

---

## Dataset Description
The dataset includes customer-related information such as:
- Customer ID
- Gender
- Senior Citizen Status
- Tenure
- Phone Service & Internet Service
- Contract Type
- Payment Method
- Monthly Charges
- Total Charges
- Churn Status

The dataset was cleaned and transformed before visualization.

---

## Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Visualization Techniques

## Dashboard Pages

### Customer Demographics Analysis
- Customer distribution by gender
- Average monthly charges by tenure
- Churn rate by tenure group
- Senior vs non-senior churn comparison

### Service Subscription Analysis
- Customer count by phone & internet service
- Churn rate by internet service type
- Most subscribed add-on services
- Impact of subscribed services on churn

### Contract & Billing Insights
- Churn by contract type
- Churn by payment method
- Average charges by contract type
- Paperless billing impact on churn
- Tenure distribution by contract type

---

## Key Insights
- Customers with month-to-month contracts have higher churn rates.
- Early-tenure customers are more likely to churn.
- Fiber optic users show higher churn compared to DSL users.
- Customers with fewer subscribed services tend to leave more often.
- Electronic check payment method shows higher churn.
- Long-term contracts help improve customer retention.

---

## Conclusion
This project highlights the major factors influencing customer churn and provides meaningful insights into customer behavior.  
By analyzing demographics, services, contract types, and billing patterns, the dashboard helps identify high-risk customers and supports better decision-making to improve customer retention.
